# 주요 기능

## 토큰 인증 로그인<br>

- 기능 : JWT 토큰 인증 방식으로 로그인 기능 구현

## CRUD

- 메인 페이지 - 전체 회원 조회 (GET)
- 상세페이지 - 나와 매치된 회원 조회 (GET), 댓글 작성 (POST)

## 이미지 업로드

- form data로 S3 서버에 데이터 전송

## '좋아요' 기능

- 좋아요 (LIKE) 버튼 기능 구현 (PATCH)
